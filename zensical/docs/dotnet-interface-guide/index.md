<h1 class="heading"><span class="name">About</span></h1>

The _.NET Interface Guide_ describes the Dyalog interface to Microsoft .NET, the cross-platform (Microsoft Windows, Linux, and macOS) successor to Microsoft's .NET Framework. It does not attempt to explain the features of .NET, except in terms of their APL interfaces.

Assumed knowledge: A familiarity with .NET and/or .NET Framework, and a basic understanding of Object-Oriented (OO) methodologies.
