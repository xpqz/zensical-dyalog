# <span class="name">Downstile</span> <span class="command">⌊</span> {: .heading}

Monadic Downstile means
[Floor](../primitive-functions/floor.md)
```apl
      ⌊ 3.4 ¯3.4 3 0

```

[Minimum](../primitive-functions/minimum.md)
```apl

1.1 ¯3.4

      ⌊/ 3 1 4 1
1
```
