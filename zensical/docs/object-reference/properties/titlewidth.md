# <span class="name">TitleWidth</span> <span class="right">Property</span> {: .heading}



**Applies To:** [Grid](../objects/grid.md)

**Description**


This property is a single number that specifies the width of the row titles displayed in a [Grid](../objects/grid.md) object. It is expressed in the units specified by the [Coord](coord.md) property of the [Grid](../objects/grid.md). The value `⍬` selects the default width.
